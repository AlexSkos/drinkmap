4) Как запускать

Клади/обновляй файлы в assets/overrides/
— можно node_647921208.jpg или любое имя с EXIF-GPS.

Запусти генерацию:
npm run build:overrides

увидишь, например:  
Map node/647921208 -> ../../assets/overrides/node_647921208.jpg
Wrote src/data/overrides.ts with 3 entries

Запускай приложение (npx expo start -c).
На карте карточки фонтанов с совпавшим ID будут показывать твои фото.
Если фото нет → используется внешний URL → если и он не открылся → fountain_defolt.jpg.
